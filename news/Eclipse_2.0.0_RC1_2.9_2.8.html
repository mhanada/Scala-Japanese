Updated: 2011-11-15, 15:08 Scala IDE for Eclipse version 2.0 のための RC1 が完成しました。 もし、重大なバグや報告がなければこれは 2.0 としてリリースします。 みなさんの忍耐と途方も無い作業のおかげです。 2.0.0 をリリースした後、2.0 の重要なバグフィックスなどのメンテナンスを続けるつもりです。 しかしながら、焦点は 2.1 の開発にシフトするつもりです。そして、新しい機能に集中するつもりです。 また、2.0.0 は 2.8 のコンパイラをサポートする最後のバージョンです。 2.8 が軽視されているわけではありませんが、これをサポートするための努力はあまりにも高く、多くの開発を減速させてしまいます。 私たちは 2.0.0 のリリースが、まだしばらく 2.9.1 にアップグレードすることができない人たち(download 全体の 2%)のためのよい IDE になると信じています。 私たちは Eclipse IDE for Scala の RC1 がダウンロード可能になったことをアナウンスできることを嬉しく思います。 このリリースは 2.9.1 version と 2.8.3-SNAPSHOT version の 2種類があります。
