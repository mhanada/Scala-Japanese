Scala LiftOff London

もし Scala と Web 開発に興味があるなら10月13,14日に行われるScala LiftOff in Londonに行くべきです。 Scala の web framework である Lift に重点をおいて、エキスパート達がどのようにすれば Scala を使って Lift の能力を最大に引き出すかを解説してくれます。 Lift createor の David Pollak による Lift についての基調講演と as always readily aveilable for unlimited discussions about Lift があります。 ほかにも、 Guardianの Graham Tackley による Lift を使うことで heavy duty media api をどのようにして提供するかや、 Ilulian Dragos による最新の Ecllipse Scala plug-in での開発、 Lift の contirubutor である Tim Perretkey の Helixでの活動、 Phil Bagwell の並行ハッシュツリーなど様々な topic があります。
